CKEditor Sample — Media (oEmbed) Plugin

This sample shows the use of the Media (oEmbed) Plugin where you can include only content from popular sites such as YouTube, Flickr, Qik, Vimeo, Hulu, Viddler and many more

CKEDITOR.replace( 'editor1', {
	extraPlugins: 'oEmbed',
	toolbar: [
		['Source', '-', 'Save','NewPage','-','Undo','Redo'],['oEmbed']
             ],
	... some other configurations omitted here
});
